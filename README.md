# Django Homework Project

Этот проект создан для выполнения домашнего задания по Django.

## Установка и запуск

1. Клонировать репозиторий
2. Создать виртуальное окружение: `python -m venv .venv`
3. Активировать окружение: 
   - Windows: `.venv\Scripts\activate`
   - Linux/Mac: `source .venv/bin/activate`
4. Установить зависимости: `pip install -r requirements.txt`
5. Применить миграции: `python manage.py migrate`
6. Запустить сервер: `python manage.py runserver`





## Ветки

- `main` - основная ветка с релизами
- `develop` - ветка разработки
- `django` - ветка для выполнения домашнего задания
